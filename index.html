I'm a software developer with a strong focus on the backend, who specializes in building clean, secure, and scalable backend systems for high-impact web and mobile products. My primary stack is Node.js and Express, with solid experience in PHP and Laravel for projects that require it. I’ve contributed to multiple platforms — from ecommerce and fintech to infrastructure tools — building features like RESTful APIs, user onboarding, authentication (JWT, Auth0), wallet automation, virtual card integration, and payment systems. I’m actively learning how to integrate AI tools and services into full-stack applications to power features like intelligent recommendations, automated decision-making, data-driven personalization and automated AI assistants.
